pub fn effective_clocks(&self) -> Vec<ClockConfig>

Return the effective clock configurations.

If clocks is set, returns it directly. Otherwise synthesizes a single-clock entry from the legacy clock_gpio + clock_period_ps fields. This ensures full backward compatibility with existing single-clock configs.
